# 毕业设计管理系统 — 页面运行说明

运行地址基准：

- `http://localhost:8086/graduation-design/`

## 启动方式

```bash
# 1. 初始化数据库（首次运行）
& "D:\MySQL\MySQL Server 8.0\bin\mysql.exe" --default-character-set=utf8mb4 -uroot -p12345 -e "source sql/init.sql"

# 2. 编译打包并启动
mvn package
mvn tomcat7:run-war
```

## 主要页面

| 页面 | 地址 | 说明 |
|------|------|------|
| 登录 | `/login.jsp` | 全屏渐变登录页 |
| 仪表盘 | `/dashboard.jsp` | 三角色首页统计 |
| 用户管理 | `/admin/users.jsp` | 管理员 CRUD |
| 公告管理 | `/admin/announcements.jsp` | 发布/置顶/删除 |
| 我的课题 | `/teacher/topics.jsp` | 教师课题管理 |
| 选题审批 | `/teacher/selections.jsp` | 批准/驳回申请 |
| 文档审核 | `/teacher/documents.jsp` | 三阶段 Tab 审核 |
| 学生进度 | `/teacher/students.jsp` | 所带学生进度 |
| 浏览课题 | `/student/topics.jsp` | 卡片式选题 |
| 我的选题 | `/student/my-selection.jsp` | 申请状态跟踪 |
| 文档提交 | `/student/documents.jsp` | 开题/中期/终稿 |
| 我的成绩 | `/student/grades.jsp` | 四阶段分数 + 答辩成绩汇总 |
| 我的答辩 | `/student/defense.jsp` | 答辩时间/教室/分组 |
| 站内消息 | `/*/messages.jsp` | 三角色收发消息 |
| 答辩安排 | `/admin/defenses.jsp` | 管理员 CRUD + Excel 导入 |
| 数据统计 | `/admin/statistics.jsp` | ECharts 图表 + Excel 导出 |
| 操作日志 | `/admin/logs.jsp` | 筛选 + 分页审计 |
| 教师答辩 | `/teacher/defense.jsp` | 所带学生答辩安排 |

## Phase 2/3 验收流程

1. **admin** → 仪表盘 ECharts → 答辩安排/Excel 导入 → 数据统计/导出 → 操作日志
2. **teacher01** → 审批选题（满员拒绝）→ 文档评阅（自动通知）→ 答辩安排查看 → 站内消息
3. **student01** → 浏览课题（已满隐藏）→ 文档上传（版本历史）→ 我的答辩 → 四阶段成绩
